# Simple HTTP Client using Python Sockets

This project implements a basic HTTP client as part of a Computer Networks assignment.

## Features
- Creates a TCP connection using Python sockets
- Sends an HTTP GET request to a server
- Receives and displays the full HTTP response
- Demonstrates basic client-server communication

## Files
- Simple_HTTP_Client.ipynb — Google Colab Notebook
- screenshots/ — Output proof for the report

## How to Run
1. Open the `.ipynb` file in Google Colab or Jupyter Notebook.
2. Run all cells.
3. Modify the host/path if needed.
